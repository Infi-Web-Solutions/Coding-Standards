# Contribution Guide

## Content Updates

- Keep titles short and scannable.
- Use the shared navbar/footer link pattern.
- Prefer tables for quick references and checklists for action items.

## Routing

- Category index pages should link to detail pages with correct relative paths.
- Pages inside `html/standards/` and `html/tasks/` do not share the same depth.

## Quality

- Validate links after creating new pages.
- Avoid leaving placeholder or empty files in committed content.
