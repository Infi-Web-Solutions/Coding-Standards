# Development Standards Website

Static standards library for team guidance, checklists, templates, and practice tasks.

## Main Areas

- `index.html`: homepage and entry point
- `html/pages/`: top-level library pages
- `html/standards/`: standards documents and category landing pages
- `html/checklists/`: delivery checklists
- `html/tasks/`: task indexes and project practice tasks
- `css/`: shared styles
- `js/`: progressive enhancement scripts

## Notes

- Pages are plain HTML and share `css/main.css`.
- Empty placeholders have been replaced with usable starter content.
- Relative links should always be checked from the current file location.
