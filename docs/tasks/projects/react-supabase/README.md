# React + Supabase Projects

> Both projects use React (with TypeScript) as the frontend and Supabase (Postgres + RLS + RPC + Storage + Edge Functions) as the backend.

| Project | What It Covers | Time |
|---------|---------------|------|
| [01 — Workspace Invite System](01-invite-system.md) | Supabase tables, RLS, Edge Function (invite email), RPC (accept invite), React invite management, token-based invite acceptance | ~8 hours |
| [02 — Document Library](02-document-library.md) | Supabase Storage, RLS on storage.objects, RPC (create folder), React file upload, signed URL download, folder navigation | ~8 hours |
